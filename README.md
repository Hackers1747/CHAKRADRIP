## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [Shopify CLI](https://shopify.dev/docs/themes/tools/cli) v3+
- A Shopify Partner account or development store

### Installation

```bash
# Clone the repo
git clone https://github.com/your-username/chakradrip.git
cd chakradrip

# Install Shopify CLI (if not already installed)
npm install -g @shopify/cli @shopify/theme

# Authenticate with your Shopify store
shopify auth login --store your-store.myshopify.com
```

### Local Development

```bash
# Start local dev server with hot reload
shopify theme dev --store your-store.myshopify.com
```

### Deploy to Store

```bash
# Push theme to your Shopify store
shopify theme push --store your-store.myshopify.com
```

## 🎨 Customization

Theme settings can be configured via the Shopify Theme Editor (`Online Store > Themes > Customize`) or by editing `config/settings_data.json` directly.

Key customizable areas:
- **Color palette** — chakra-coded accent colors per collection
- **Typography** — anime display fonts + clean body text
- **Homepage sections** — hero banners, featured drops, lookbook grid
- **Product cards** — hover animations, energy aura effects
